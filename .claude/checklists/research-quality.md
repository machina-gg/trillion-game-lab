# リサーチ品質チェックリスト

Researcher がリサーチ完了時に確認する。

## データの信頼性

- [ ] 全てのデータにソースURLが記載されている
- [ ] 重要な主張は2つ以上のソースで裏付けている
- [ ] データの取得日が明記されている
- [ ] ソースの信頼度（★★★/★★/★）を評価している

## 市場分析

- [ ] TAM/SAM/SOM の推定がある
- [ ] 成長率（CAGR）のデータがある
- [ ] 主要プレイヤーが3社以上リストされている

## トレンド

- [ ] 直近6ヶ月のトレンドデータがある
- [ ] テクノロジートレンドと消費者トレンドの両方をカバー

## 競合分析

- [ ] 競合が3社以上分析されている
- [ ] 機能比較マトリックスがある
- [ ] Gap 分析でブルーオーシャン機会を特定している

## ソース一覧

- [ ] sources.md に全URLが一覧化されている
- [ ] 一次ソースと二次ソースが区分されている
