# 企画書品質チェックリスト

Planner が企画書完了時に確認する。

## コンセプト

- [ ] エレベーターピッチが1-2文で明確に書けている
- [ ] 解決する課題が具体的に定義されている
- [ ] ターゲットユーザーが明確

## 差別化

- [ ] 競合分析データに基づいた差別化
- [ ] 「なぜ既存プロダクトではダメか」が説明できている
- [ ] 差別化が持続可能か検討されている

## MVP

- [ ] 機能が3-5個に絞られている
- [ ] 各機能の説明が具体的
- [ ] MVP で最小限の価値検証ができる

## ビジネスモデル

- [ ] 収益化の方法が記載されている
- [ ] 市場規模データに基づく収益予測

## リサーチ参照

- [ ] research-ref.md にリサーチへのリンクがある
- [ ] 企画書内のデータがリサーチと一致している
